# Research

Per-product research files live here. They are **gitignored** — only `research/README.md` is committed.

## Files agents write here

| File | Written by | Purpose |
|---|---|---|
| `competitor-analysis.md` | Tanmay (Stage 1) | Competitor ad breakdown — formats, hooks, CTAs, spend estimates |
| `winning-hooks.md` | Tanmay (Stage 1) | Hook patterns ranked by frequency + effectiveness |
| `audience-insights.md` | Tanmay (Stage 1) | ICP language, objections, pain points from Reddit/Quora/YouTube |

## How to start

1. Fill `state/product-context.md` (copy from `state/product-context.md.template`).
2. Ask Zimmer to kick off Cycle 1 Research (Stage 1).
3. Tanmay will write all three files above.

## Ad library data (optional)

If you export raw data from the Meta Ad Library or similar tools, place CSVs/JSONs under `research/ad-library-data/`. They are gitignored.
